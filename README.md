# ⚛️ React

React is a **JavaScript library** for building fast, dynamic, and reusable **user interfaces (UIs)**.  
It was developed by **Facebook** and is widely used for building **single-page applications (SPAs)** with a smooth user experience.

---

## 🚀 Why React?
- ⚡ **Fast Rendering** – Uses Virtual DOM for high performance  
- 🧩 **Component-Based** – Build reusable and modular UI parts  
- 🔄 **One-Way Data Flow** – Predictable and easy-to-debug applications  
- 🌍 **Huge Ecosystem** – Rich community, libraries, and tools  
- 📱 **Cross-Platform** – Supports web, mobile (React Native), and more  

---

## 🛠️ Core Concepts
- **Components** → Small, reusable UI building blocks  
- **Props** → Pass data from parent to child component  
- **State** → Manage data that changes over time  
- **JSX** → Write HTML-like code inside JavaScript  
- **Hooks** → Use state and lifecycle features in functional components
